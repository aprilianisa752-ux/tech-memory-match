# 🎮 Tech Memory Match

Tech Memory Match adalah permainan memory card bertema teknologi yang dibuat menggunakan HTML, CSS, dan JavaScript. Project ini dijalankan menggunakan Docker dan di-deploy pada VPS.

## ✨ Features

- Memory card game
- Responsive design
- Timer
- Move Counter
- Restart Game
- Technology-themed icons

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Docker
- Nginx
- Ubuntu VPS
- Git & GitHub

## 📂 Project Structure

```
tech-memory-match/
│
├── css/
├── js/
├── index.html
├── Dockerfile
└── README.md
```

## 🚀 Running with Docker

Build image

```bash
docker build -t tech-memory-match .
```

Run container

```bash
docker run -d -p 8080:80 tech-memory-match
```

Open browser

```
http://localhost:8080
```
